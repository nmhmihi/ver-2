# **App Name**: Currency Clarity

## Core Features:

- Exchange Rate Setup: Allows users to input and store the current exchange rate between TWD and VND.
- Detailed Transaction Management: Allows users to add new transactions by entering the sender's name, amount in TWD, and any fees (in VND). The application automatically calculates the equivalent VND amount after deducting fees.
- Transaction Editing: Allows the user to edit the the fields of each transaction such as sender's name, TWD amount, fees, etc.
- Transaction Deletion: Allows users to delete transactions that are incorrect or no longer needed.
- Transaction Listing: Displays all transactions in a clear list, sorted by time (newest first), showing sender's name, TWD amount, fees, converted VND amount, and transaction time. Calculates and displays the total VND amount of all transactions.
- Daily Revenue Summary: Summarize total converted VND amount for each day, so users can easily monitor daily exchange performance and identify trends.
- Overall Currency Totals: Show totals of all TWD and VND from all transactions, with each currency amount displayed in a separate column for easy comparison and overview of cash flow.

## Style Guidelines:

- Primary color: Saturated purple (#9D39B3) to convey trust and sophistication. The purple evokes currency and business.
- Background color: Light gray (#F0F0F5), a desaturated version of the purple primary, for a clean and neutral backdrop.
- Accent color: Blue (#397AB3) to complement the primary purple and provide clear visual cues.
- Headline font: 'Poppins' (sans-serif), giving the app a geometric, fashionable look, mainly used for headlines.
- Body font: 'PT Sans' (sans-serif), for all main content for readability.
- Use minimalist, consistent icons for actions and navigation to maintain clarity and ease of use.
- A clean, well-spaced layout to present data clearly. Important information like totals and exchange rates are prominently displayed.
